
## Installation for Barnyard2

Install Barnyard2 for Unified2 File Reading

Unpack, change directories, configure with mysql support, make and install. 

```bash
cd distro
tar -zxvf barnyard2*.tar.gz
cd barnyard2*
./configure --with-mysql
make
sudo make install
```
